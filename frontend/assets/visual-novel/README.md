Visual novel asset drop-in guide

Add your new story art under these folders:

- `frontend/assets/characters/lea/`
- `frontend/assets/characters/mara/`
- `frontend/assets/characters/drlin/`
- `frontend/assets/backgrounds/`

Expected character files:

- `frontend/assets/characters/lea/base.png`
- `frontend/assets/characters/lea/worried.png`
- `frontend/assets/characters/lea/determined.png`
- `frontend/assets/characters/lea/relieved.png`
- `frontend/assets/characters/lea/reflective.png`
- `frontend/assets/characters/mara/base.png`
- `frontend/assets/characters/mara/composed.png`
- `frontend/assets/characters/mara/persuasive.png`

- `frontend/assets/characters/drlin/warm.png`
- `frontend/assets/characters/drlin/informative.png`
